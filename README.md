# dsgrid
Here are the steps to install dsgrid after cloning the repository.

## Users

```
pip install -e .
```

## Developers

```
pip install -e '.[dev]'
```

## Conda environment
A conda environment is available for your convenience.

```
conda env create -f environment.yml

pip install -e .

# or

pip install -e '.[dev]'
```

## Usage

### List the available commands.

```
dsgrid --help
```
